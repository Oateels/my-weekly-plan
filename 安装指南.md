# 周计划 · 安装与同步指南

这份指南教你 5 分钟完成设置，之后就能在多台设备间自动同步数据。

---

## 第一步：注册 GitHub 账号（如果还没有）

1. 打开 [github.com](https://github.com)
2. 点右上角 **Sign up**
3. 填邮箱、密码、用户名 → 完成注册
4. 去邮箱点验证链接

> 免费账号即可，不需要付费。

---

## 第二步：创建自己的数据仓库

1. 登录后，点右上角 **+** → **New repository**
2. 仓库名填 `my-weekly-data`（随便起，但建议这个）
3. 选择 **Private**（私人仓库，别人看不到你的数据）
4. 其他选项不用动，直接点 **Create repository**

⏸️ **先别关这个页面，等一下要用。**

---

## 第三步：创建访问令牌（Token）

1. 点右上角头像 → **Settings**
2. 左侧菜单拉到最下面 → **Developer settings**
3. 左侧点 **Personal access tokens** → **Tokens (classic)**
4. 点 **Generate new token** → **Generate new token (classic)**
5. 在 Note 里填 `周计划同步`（方便以后认出来是谁）
6. 过期时间选 **No expiration**
7. 勾选以下权限：
   - ☑️ **repo**（勾这一个，下面的子项会自动全选）
8. 拉到页面最下面，点 **Generate token**
9. ⚠️ **复制出现的绿色 token 码，立刻存到备忘录里！** 关掉后就再也看不到了。

---

## 第四步：在周计划里填入信息

1. 打开 [oateels.github.io/my-weekly-plan/](https://oateels.github.io/my-weekly-plan/)
2. 点右上角 ⚙️ **设置**
3. 填入以下两行：

| 输入框 | 填什么 |
|--------|--------|
| GitHub Token | 粘贴你第三步复制的 token 码 |
| 仓库名 | `你的用户名/my-weekly-data`（比如 `xiaoming/my-weekly-data`） |

4. 点 **☁️ 立即同步**

看到「已同步」三个字就搞定了 ✨

---

## 其他设备怎么连

在新手机/新电脑上：

1. 打开同一个链接
2. 点 ⚙️ 设置 → 填入 **相同的 Token 和仓库名**
3. 点 ☁️ 立即同步

所有数据自动同步过去了。

---

## 常见问题

**Q：Token 忘了怎么办？**
去 GitHub Settings → Developer settings → Tokens → 删掉旧的，新建一个就行。

**Q：我不想用同步，能纯本地用吗？**
可以。不填 Token，所有数据存在手机/电脑浏览器里。但换设备就找不到数据了。

**Q：我的数据安全吗？**
仓库选了 Private 就没人能看到。Token 也只存在你自己的浏览器里。
